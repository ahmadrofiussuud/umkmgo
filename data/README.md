# Folder `/data`

Folder ini berisi file data statis atau dummy (dalam format JSON atau TypeScript) yang digunakan oleh aplikasi.

## Aturan Penggunaan
- Simpan data tiruan/mocking di folder ini untuk memisahkan logika UI dengan data mentah.
- Membantu kemudahan migrasi jika nantinya data dipindahkan ke database asli atau Headless CMS.
