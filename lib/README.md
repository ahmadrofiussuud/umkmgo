# Folder `/lib`

Folder ini berisi fungsi utilitas, helper, API client, dan konfigurasi pustaka pihak ketiga yang digunakan oleh aplikasi.

## Aturan Penggunaan
- Contoh isi folder ini adalah fungsi formatting mata uang, helper WhatsApp generator, atau konfigurasi database/CMS client.
- Buat file helper modular (contoh: `utils.ts`, `whatsapp.ts`).
