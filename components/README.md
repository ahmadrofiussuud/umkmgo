# Folder `/components`

Folder ini berisi komponen UI modular dan reusable yang digunakan di berbagai halaman aplikasi.

## Aturan Penggunaan
- Simpan komponen global di tingkat root `/components`.
- Kelompokkan komponen yang spesifik untuk fitur tertentu jika diperlukan (misalnya: `/components/umkm`, `/components/layout`).
- Gunakan TypeScript dan ikuti pola penamaan PascalCase (contoh: `CardProduct.tsx`, `Header.tsx`).
