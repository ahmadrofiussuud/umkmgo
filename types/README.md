# Folder `/types`

Folder ini berisi definisi tipe data dan interface TypeScript yang digunakan secara global di seluruh aplikasi.

## Aturan Penggunaan
- Semua model data (seperti UMKM, Produk, User, dll.) harus didefinisikan tipenya di sini.
- Membantu menjaga konsistensi tipe data (type safety) di seluruh file komponen maupun helper.
